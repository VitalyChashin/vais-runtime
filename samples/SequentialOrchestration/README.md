# SequentialOrchestration

Three participants piped: researcher → writer → editor. Each sees the prior participant's output as its user message. Scripted providers for determinism.

**Concepts:** [orchestration](../../docs/concepts/orchestration.md).
**Packages:** `Vais.Agents.Abstractions`, `Vais.Agents.Core`.
**Needs API key:** no.

```bash
dotnet run --project samples/SequentialOrchestration
```
