# InputOutputGuardrails

Prompt-injection guardrail on input + credit-card-regex guardrail on output. Shows three scenarios: input trip, output trip, clean pass.

**Concepts:** [guardrails](../../docs/concepts/guardrails.md).
**Packages:** `Vais.Agents.Abstractions`, `Vais.Agents.Core`.
**Needs API key:** no.

```bash
dotnet run --project samples/InputOutputGuardrails
```
