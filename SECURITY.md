# Security

## Supported versions

This project is in Phase 1 (pre-alpha). No versions are under long-term security support yet. Use at your own risk.

## Reporting a vulnerability

Please report security issues **privately** to the project maintainers — do not open a public issue. Once a stable contact channel and advisory process exists, it will be documented here.

Expected response: an acknowledgement within 5 working days and a plan within 10. Vulnerabilities in transitive dependencies should be reported upstream first; we track a Dependabot/Renovate feed.
