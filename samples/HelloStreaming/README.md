# HelloStreaming

Basic `StatefulAiAgent.StreamAsync` with a scripted provider. Deltas print character-by-character; session holds the final assistant turn.

**Concepts:** [execution loop](../../docs/concepts/execution-loop.md).
**Packages:** `Vais.Agents.Abstractions`, `Vais.Agents.Core`.
**Needs API key:** no.

```bash
dotnet run --project samples/HelloStreaming
```
