# RoundRobinOrchestration

Two participants debate for up to 3 rounds. `TerminationPredicate` stops on agreement. Each sees the shared conversation with prior steps encoded as assistant turns.

**Concepts:** [orchestration](../../docs/concepts/orchestration.md).
**Packages:** `Vais.Agents.Abstractions`, `Vais.Agents.Core`.
**Needs API key:** no.

```bash
dotnet run --project samples/RoundRobinOrchestration
```
